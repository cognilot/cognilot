# Epic 07: Productivity Analytics (Futuro)

## Contexto

Documenta el valor tangible que Cognilot aporta al usuario mediante métricas de ahorro de tiempo y esfuerzo.

## Funcionalidades (User Stories)

### F7.1 - Contador de Tiempo Ahorrado

- **Descripción:** Un algoritmo que estima cuántos minutos ha ahorrado el usuario basándose en la cantidad de caracteres autocompletados.
- **Criterio de Aceptación:** Visualización en el dashboard (`/welcome`) de una cifra acumulativa (ej. "Has ahorrado 45 minutos este mes").

### F7.2 - Registro de Formularios Completados

- **Descripción:** Historial visual de cuántas veces se ha activado la IA con éxito.

### F7.3 - Mapa de Eficiencia

- **Descripción:** Identificar en qué tipos de formularios o sitios web el usuario es más productivo gracias a la herramienta.

## Reglas de Negocio

- **Privacidad Total:** Las métricas son anónimas y solo cuentan "caracteres" y "eventos", nunca el contenido sensible de lo que se llenó.
